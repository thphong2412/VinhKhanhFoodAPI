Actors & Components:
• Owner (Chủ quán) — Người dùng truy cập OwnerPortal để đăng ký/đăng nhập
• OwnerPortal — Razor Pages web app (Register.cshtml.cs, Login.cshtml.cs)
• AuthController — API: admin/auth/register-owner, admin/auth/login
• Database — Tables: Users (email, passwordHash, isVerified), OwnerRegistrations (status, shopName, cccdEncrypted)
• SyncHub (SignalR) — Gửi event "OwnerRegistrationSubmitted" realtime
• AdminPortal — Admin nhận thông báo và duyệt đơn đăng ký
⚠️ Bảo mật: Password được hash bằng SHA256 + static salt (POC). CCCD được mã hoá bằng EncryptionService.Protect. JWT token có thời hạn 180 phút.
